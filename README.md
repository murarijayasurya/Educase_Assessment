# Educase Assessment

This project is a React JS implementation of the Educase mobile UI assessment. It reproduces the provided Adobe XD design with a centered mobile-style layout, clean component structure, and seamless page navigation.

## Project Overview

- Built with React JS and Vite
- Styled with Tailwind CSS
- Mobile-first interface centered on the page
- Navigation between the Home, Sign Up, Login, and Account pages
- Clean, readable, and easy-to-maintain component code

## Design Reference

- Adobe XD: https://xd.adobe.com/view/b68eea25-003d-4a5d-8fdd-d463eeb20b32-e3dd

## How to Run

1. Install dependencies.
   ```bash
   npm install
   ```
2. Start the development server.
   ```bash
   npm run dev
   ```
3. Build for production.
   ```bash
   npm run build
   ```

## Deployment

- GitHub repository: https://github.com/murarijayasurya/Educase_Assessment
- Live demo: https://educase-assessment-three.vercel.app/


## Notes

- The layout is centered to match the mobile app presentation from the design.
- Routes are handled with React Router for smooth page-to-page navigation.
- Form fields and account screen content are structured to match the assessment flow.
